classless

A collection of themes for blog and blog-like websites (anything with pages/posts and sections/categories) based on pure semantic HTML.

All themes follow the same HTML structure, so you can just swap the CSS (sometimes JS) and you get a whole new theme without having to change anything. The HTML consists of only tags, no classes or ids, and makes very little assumptions about the content with which you will fill it.

(see this in HTML)

For theme writers

Write your theme once, constrain yourself a little to the limited customizationability of the HTML, but magically you get compatibility with all CMSes out there.

Perhaps, however, for doing nice stuff you'll need to use tricky CSS.

For CMS builders

If you're writing a CMS, be it a hosted service with a specific property or an open-source Wordpress alternative you don't need to think about themes (and all the network effect arising from it: people use if we have beautiful themes, theme builders come if we have a users) or try to be compatible with other CMSes to use themes designed for them specifically. You can just be compatible with HTML5 sections and comply to this minimal standard and you'll get automatically our pure CSS (and JS sometimes) open source themes.

You also free yourself from the burden of writing a mechanism that allow theme builders to write HTML, share it (maybe even have a "theme market") and get that HTML rendered by your software, you can accept just CSS and JS, which can be just included in the rendered page and that's it.

Contribute!

As you can see, I'm not the best designer in the world, and all these themes are ports from random themes open-sourced on the web, so help with theme creation and suggestions for improvements in the basic template will be very much appreciated.

There's a GitHub repository with the basic template and all themes, and a playground for dinamically testing themes with various different HTML documents based on the fundamental structure.
