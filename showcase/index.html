classless

A collection of themes for blog and blog-like websites (anything with pages/posts and sections/categories) based on pure semantic HTML.

All themes follow the same HTML structure, so you can just swap the CSS (sometimes JS) and you get a whole new theme without having to change anything. The HTML consists of only tags, no classes or ids, and makes very little assumptions about the content with which you will fill it.
